I have a Goals page in my LogSeq vault. It's the thing everything else references -- the [weekly review](/life-os/weekly-review) checks progress against it, [morning check-in](/life-os/morning-checkin) priorities should trace back to it, [evening check-in](/life-os/evening-checkin) asks what got done.

In-person over apps. Building out a vetting framework and tracking weekly dating activity.

## Template

```markdown
# Goals
The big three. Everything else supports these.

## 1. [Goal Name] — [Target]
**Current:** [where you are now]
**2026 Target:** [specific measurable target]
**Ultimate:** [longer-term version if applicable]
**Key Insight:** [the one thing that matters most for progress]

### Subgoals
- [ ] [Milestone 1] — [status]
- [ ] [Milestone 2] — [status]
- [ ] [Milestone 3] — [status]

---

## Weekly Review Checklist
- [ ] Check progress on each goal's subgoals
- [ ] Update any blockers or changes
- [ ] Set focus for coming week
```

## Setup Prompt

```
Create a Goals page in my vault. Use type:: system so it's
distinct from notes and reference material.

I want to track 3 major goals. For each one, include:
- The goal name and target
- Current state (where I am now)
- 2026 target (specific and measurable)
- Key insight (the one lever that matters most)
- Subgoals as checkboxes
- Any supporting context

At the bottom, add a weekly review checklist.

My three goals are:
1. [Goal 1]
2. [Goal 2]
3. [Goal 3]
```

## How It Connects

The goals page feeds into the rest of the Life OS:

- **[Weekly Review](/life-os/weekly-review)** -- has a "Goal progress" section. The subgoal checkboxes give you something concrete to check in on.
- **[Monthly Review](/life-os/monthly-review)** -- zooms out on each goal over the past month.
- **[Morning Check-in](/life-os/morning-checkin)** -- if none of your priorities for the day relate to your goals, worth noticing.
- **[Evening Check-in](/life-os/evening-checkin)** -- "what got done today" is a daily goal progress check.

Goals you don't review are goals you forget about. The weekly review checklist at the bottom is there for that reason.
