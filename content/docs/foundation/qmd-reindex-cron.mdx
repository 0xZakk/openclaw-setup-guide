---
title: Nightly QMD Reindex
description: Keep search indexes fresh with a nightly reindex cron job
---

Runs `qmd update` and `qmd embed` every night at 2 AM to reindex all search collections. New vault notes, memory files, and workspace changes all become searchable the next day.

## Setup Prompt (Cron Job)

```
Create a cron job called "Nightly QMD Reindex" at 2:00 AM every day.

Run these commands:
  qmd update
  qmd embed

This updates all collections and regenerates embeddings for any
new or changed files. Log results (file counts, any errors) in
today's memory file. If there are errors, message me. Otherwise
complete silently.
```

- **Schedule:** `0 2 * * *` (2:00 AM daily)
- **Timeout:** 600 seconds (embedding can be slow with large vaults)
