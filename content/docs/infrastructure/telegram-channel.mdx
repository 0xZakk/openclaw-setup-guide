## What It Does

Telegram is the primary messaging surface -- how you talk to your agent from your phone or desktop. Messages you send on Telegram go to the agent; the agent's replies come back to Telegram.

## Setup

1. Create a Telegram bot via [@BotFather](https://t.me/botfather)
2. Get the bot token
3. Add to OpenClaw config:

```json
{
  "channels": {
    "telegram": {
      "enabled": true,
      "botToken": "your-bot-token",
      "dmPolicy": "pairing",
      "groupPolicy": "allowlist",
      "streamMode": "partial"
    }
  }
}
```

4. Start a chat with your bot on Telegram
5. OpenClaw will prompt you to pair (approve the pairing)

## Configuration Options

- **dmPolicy: "pairing"** -- requires pairing approval for DMs (recommended)
- **groupPolicy: "allowlist"** -- only responds in approved groups
- **streamMode: "partial"** -- streams partial responses as the agent types

## Features

- Voice messages are transcribed automatically (if audio transcription is configured)
- Images are analyzed via vision models
- Files can be sent and received
- Inline buttons for interactive prompts
- Reactions on messages
