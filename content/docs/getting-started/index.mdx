## Prerequisites

- OpenClaw installed and running (`npm install -g openclaw`)
- A messaging channel connected (Telegram, Discord, Signal, etc.)
- An Anthropic API key configured
- A machine that stays on (Mac, Linux server, Raspberry Pi)

## First Steps

After installing OpenClaw and connecting your messaging channel, the first thing to do is give your agent an identity and teach it about you. This is what makes it *yours* rather than a generic chatbot.

1. **Run the onboarding wizard:** `openclaw onboard`
2. **Set up your workspace files** -- see [Agent Identity](/docs/foundation/agent-identity), [User Profile](/docs/foundation/user-profile), and [Memory System](/docs/foundation/memory-system)
3. **Add your first cron job** -- start with a [Morning Check-in](/docs/life-os/morning-checkin) to see daily value immediately
4. **Layer on from there** -- see [Minimum Viable Setup](/docs/getting-started/minimum-viable-setup) for a phased rollout

## How This Guide Works

Each page in this guide describes:
- **What it does** -- the system/workflow/skill explained
- **Setup Prompt** -- a prompt you can paste into your agent's chat to set it up
- **Configuration** -- any config changes or external tools needed

Adapt everything to your own preferences. The prompts are starting points, not rigid scripts.
