## Prerequisites

- OpenClaw installed (`npm install -g openclaw`)
- A messaging channel connected (Telegram, Discord, Signal, etc.)
- An Anthropic API key
- A machine that stays on (Mac, Linux server, Raspberry Pi -- whatever you've got)

## First Steps

Once OpenClaw is installed and connected to your messaging channel, the first thing to do is give your agent an identity and teach it about you. This is what makes it *yours* instead of a generic chatbot.

1. **Run the onboarding wizard:** `openclaw onboard`
2. **Set up your workspace files** -- [Agent Identity](/foundation/agent-identity), [User Profile](/foundation/user-profile), and [Memory System](/foundation/memory-system)
3. **Add your first cron job** -- I'd start with a [Morning Check-in](/life-os/morning-checkin). You'll see daily value immediately.
4. **Layer on from there** -- check the [Minimum Viable Setup](/getting-started/minimum-viable-setup) for a phased rollout

## How This Guide Works

Each page covers one piece of the system:
- A description of what it does and why I set it up
- A **setup prompt** you can paste into your agent's chat to get it going
- Any config changes or external tools you'll need

Everything here is a starting point. Adapt it to how you actually work.
