---
title: Minimum Viable Setup
description: A phased rollout to get the most value in the least time
---

Don't try to set everything up at once. I built this over weeks. Here's how I'd do it if I were starting fresh.

## Phase 1 -- Identity and Memory (Day 1)

The foundation. Without these, your agent has no continuity between sessions and no idea who you are.

- [Agent Identity](/foundation/agent-identity) -- name, personality, operating instructions
- [Memory System](/foundation/memory-system) -- daily logs + long-term curated memory
- [User Profile](/foundation/user-profile) -- who you are, your goals, preferences

About 30 minutes of prompting and iterating.

## Phase 2 -- Life OS (Day 2)

This is where you start getting daily value.

- [PARA Organization](/foundation/para-organization) -- give the workspace some structure
- [Morning Check-in](/life-os/morning-checkin) -- daily planning template
- [Evening Check-in](/life-os/evening-checkin) -- daily reflection template
- [Heartbeat System](/automation/heartbeat-system) -- proactive background check-ins

About 20 minutes to set up the cron jobs. After this, your agent is showing up for you every day.

## Phase 3 -- Knowledge Management (Week 1)

This is where it gets really interesting.

- [LogSeq/Obsidian Vault](/knowledge-management/logseq-vault) -- your personal knowledge base
- [Zettelkasten Skill](/knowledge-management/zettelkasten-skill) -- process any source into atomic notes
- [QMD Search](/knowledge-management/qmd-search) -- semantic search across everything

About an hour for initial setup, then it grows as you feed it sources.

## Phase 4 -- Content and Development (Week 2+)

Power-user stuff. Each of these is independent -- pick what's relevant to you.

- [Voice Profile](/foundation/voice-profile) -- capture your writing voice for ghostwriting
- [Tweet Pipeline](/content-creation/tweet-pipeline) -- turn vault notes into tweets
- [Coding Agent](/development/coding-agent) -- delegate coding to sub-agents
- Infrastructure: [Brave Search](/infrastructure/brave-search), [Audio Transcription](/infrastructure/audio-transcription), [1Password](/infrastructure/onepassword)
