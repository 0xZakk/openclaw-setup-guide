## What It Does

A skill that takes a draft post and improves it using research-backed guidelines, your top-performing posts, topic-specific winners, and current trends. Produces 5 rewrite variations to choose from.

## Setup Prompt

```
Create an X writing system skill. When I give you a draft post:

1. Analyze it against writing guidelines (Matt Gray style: hook,
   value, CTA patterns)
2. Compare against my top-performing posts (reference VOICE.md)
3. Search for topic-specific winners on X
4. Check current trends related to the topic
5. Produce 5 rewrite variations, each with a different angle

The rewrites should sound like ME, not like a content strategist.
Use VOICE.md as the voice filter for every variation.
```

## When to Use

- **Tweet pipeline** -- for daily low-effort drafts from vault notes
- **X writing system** -- when you have a specific draft you want to optimize
